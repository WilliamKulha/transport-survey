# _Transport Survey_

#### _A basic program that uses jQuery to gather user input from checkboxes learnhowtoprogram.com, 10-1-2018_

#### By _**William Kulha**_

## Description

_An application that takes the user's responses to a survey about transport methods and displays them once the survey is submitted_

## Setup/Installation Requirements

* _Download or clone this repo <code>$git clone https://github.com/zangiboy/transport-survey.git</code>_
* _Navigate to the folder where you saved this repository._
* _Open index.html with the browser of your choice and enjoy_
